# Mac用户键盘导航指南

## 🎯 Mac电脑键盘快捷键说明

### 为什么没有Alt键？
Mac电脑没有传统的`Alt`键，取而代之的是`Option`键（⌥）。在我们的网站中，您可以使用以下键位：

### 辅助功能模式快捷键

| Windows快捷键 | Mac快捷键 | 功能 |
|---------------|-----------|------|
| `Alt + A` | `⌥ Option + A` 或 `⌘ Cmd + A` | 开启/关闭辅助功能模式 |
| `Alt + 1-9` | `⌥ Option + 1-9` 或 `⌘ Cmd + 1-9` | 快速跳转到对应序号的帖子 |

### 其他键盘快捷键

| 快捷键 | Mac等效键 | 功能 |
|--------|-----------|------|
| `Ctrl + K` | `⌘ Cmd + K` | 聚焦搜索框 |
| `Ctrl + P` | `⌘ Cmd + P` | 收藏/取消收藏帖子 |
| `Ctrl + G` | `⌘ Cmd + G` | 跳转到第N个帖子（输入框） |
| `Ctrl + +` | `⌘ Cmd + +` | 创建新帖子 |
| `?` | `?` 或 `Shift + /` | 显示键盘帮助 |

## 🚀 如何使用

### 1. 开启辅助功能模式
- 在社区页面，按 `⌥ Option + A` 或 `⌘ Cmd + A`
- 或者点击右下角的蓝色"A"按钮

### 2. 查看快捷键提示
- 开启辅助模式后，每个帖子标题会显示"Alt+1"、"Alt+2"等提示
- Mac用户请理解为 `⌥ Option + 1`、`⌥ Option + 2` 等

### 3. 快速导航
- 按 `⌥ Option + 数字` 或 `⌘ Cmd + 数字` 直接跳转到对应帖子
- 例如：`⌥ Option + 1` 跳转到第一个帖子

### 4. 获取帮助
- 按 `?` 键打开键盘帮助对话框
- 会显示Mac用户专用的快捷键说明

## 💡 Mac键盘布局说明

```
Mac键盘上的特殊按键：
⌘ Cmd (Command) - 相当于Windows的Ctrl
⌥ Option - 相当于Windows的Alt
⌃ Control - Mac的控制键
```

## 🎨 视觉指示

- **辅助模式开启时**：右下角"A"按钮会高亮显示
- **快捷键提示**：帖子标题旁显示蓝色的快捷键标签
- **键盘帮助**：显示Mac用户专用的快捷键说明

## 🔧 故障排除

如果快捷键不工作：
1. 确保辅助功能模式已开启
2. 尝试使用 `⌘ Cmd` 键代替 `⌥ Option` 键
3. 检查是否有其他程序占用了这些快捷键
4. 刷新页面重新加载设置

## 📞 技术支持

如果您在Mac上遇到问题，请查看浏览器控制台的错误信息，或联系技术支持。
